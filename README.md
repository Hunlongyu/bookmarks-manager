<h1 align="center">Bookmarks Manager 书签管理器</h1>


### ✨ Features 特性

- 🍕 全平台支持。Windows, Mac, Linux.
- 🍥 检测重复链接。Detect duplicate links.
- 🍔 检测无效链接。Detect invalid links.
- 🍟 支持批量操作。Batch operation is supported.
- 🍤 支持多语言。Support multiple languages.(中文，English)
- 🍣 编辑书签。 Edit bookmark.
